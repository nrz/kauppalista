# Kauppalista
Useamman käyttäjän yhteinen, helposti päivitettävä selainpohjainen kauppalista.
Käyttäjillä voi olla useampi ostoslista käytössä ja kaikkiin ostoslistoihin voi liittyä useampi käyttäjä.
